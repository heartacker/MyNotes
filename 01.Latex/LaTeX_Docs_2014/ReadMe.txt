LaTeX 是一个面向作者的排版准备系统，是科技排版事实上的标准。

网络上和图书市场关于 LaTeX 的介绍文档和书籍很多，但质量层次不齐。又因为 LaTeX 发展的很快，特别是关于中文处理的部分，因此许多文档和书籍在现在看来都是过时的。比如认为「LaTeX 只支持 eps 格式的图档」，或者「推荐使用 CJK/CCT/天元支持中文」就都是老黄历了。

为了解决这个问题，为后来者铺路，同时也作为我使用 LaTeX 两年多的总结，我制作了这样一份靠谱的 LaTeX 文档压缩包。

* 推荐按照编号的顺序阅读其内的文档；
* 每个目录下都有足够详细地说明文稿，希望各位能在阅读 PDF 文档之前先阅读一下。
