「LaTeX Notes」是水木社区 TeX/LaTeX 版轰动一时的笔记，作者是自称「包太雷」的黄新刚。这份笔记以略带调侃的语气，将 LaTeX 常用的功能娓娓道来，是中文 LaTeX 社区不可多得的优秀作品。

「lshort」是经典的 LaTeX 入门材料，为许多人所推荐。压缩包中包含的是它的中文译本。lshort 介绍的内容不包括中文处理的部分，因此对于中文用户来说它的信息量略显不足。此外由于年代久远，其部分内容也与当前 LaTeX 的发展有些脱节。但它仍然是足够优秀的入门材料。

----------

在「新手请先读我」中，刘海洋介绍了关于 TeX 家族的一些情况，讲解了哪些知识容易过时哪些相对稳定。因为 LaTeX 的中文支持方式的快速发展，中文图书市场关于 LaTeX 的书籍在这个方面——也是很多人特别在意的方面——已经过时。比如陈志杰等人编写的《LaTeX 入门与提高》中介绍的中文处理部分，就是如此。

中文 LaTeX 社区最新且权威的入门书籍当属刘海洋的《LaTeX 入门》一书。这本书从最基本的 LaTeX 入门（Hello World）讲起，涉及到的内容不仅适用于新手入门，还包含许多进阶内容。值得一提的是这本书的行文风格与 TeX 作者高德纳教授的 The TeXbook 有些相近，不失为一个亮点。

《LaTeX 入门》于 2013 年 6 月出版，但截至今时已有盗版 PDF 流出。就我个人的看法，这本书的内容绝对超值于它的定价。因此出于对中文 LaTeX 社区前辈的尊重、对于版权的重视、对于书中介绍 LaTeX 的知识的推崇，这份压缩包并不包含这本书的 PDF 电子版。十分建议有需要的朋友购买纸质版，支持和尊重作者的工作（以及中文 LaTeX 社区）。当然 PDF 电子版优势在于方便，对于知识的传递的功劳不可否认。
